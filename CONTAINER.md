# teleinfo2mqtt Container Images

Container images based on [teleinfo2mqtt](https://github.com/fmartinou/teleinfo2mqtt), a service that reads Linky smart meter data (T├®l├®-Information Client) and publishes it to an MQTT broker.

Sources are available on [GitHub](https://github.com/anthochamp/container-teleinfo2mqtt).

See [README.md](README.md) for full documentation and configuration reference.

## Image tags

- `x.y.z-teleinfo2mqttA.B.C`: Container image version `x.y.z` with teleinfo2mqtt `A.B.C`.
- `edge-teleinfo2mqttA.B.C`: Latest commit build with teleinfo2mqtt `A.B.C`.
